    第一章 随机事件与概率¶

    记笔记

    这是整个学科的基石，核心在于“如何量化随机性”

    一、 知识地图与逻辑关系¶

    主线： 随机试验 → 样本空间 → 随机事件 → 事件的关系与运算 → 概率的定义与性质 → 条件概率 → 两大公式（乘法、全概率、贝叶斯） → 独立性

    关键逻辑：

    1. 概率是赋予事件的一个满足公理的数值度量。
    2. 条件概率是在“信息更新”后对概率的重新评估。
    3. 独立性是事件间一种特殊的概率关系，它简化了计算。

    二、 核心知识点精讲与易错点¶

    1. 基本概念¶

    1.1.随机试验 (E)： 可重复、结果明确、事先不可预知。

    1.2.样本空间 (Ω)： 所有可能结果的集合。

    1.3.随机事件 (A, B, C...)： 样本空间的子集。

    1.4.事件的关系与运算（用集合语言理解）

    • 包含 (A⊂B)：A发生必导致B发生。

    • 相等 (A=B)：A⊂B且B⊂A。

    • 和/并 (A∪B)：A与B至少一个发生。

    • 积/交 (A∩B 或 AB)：A与B同时发生。

    • 差 (A-B)：A发生但B不发生。A-B = A ∩ B̄

    • 互斥/互不相容 (A∩B=Ø)：A与B不能同时发生。

    • 对立/逆 (Ā)：“A不发生”。核心性质：A∪Ā=Ω, A∩Ā=Ø。

    运算律：交换、结合、分配律，以及德摩根律：A∪B = Ā ∩ B̄， A∩B = Ā ∪ B̄ 。常考：用德摩根律化简复杂事件表达式。

    2. 概率的定义与性质¶

    • 公理化定义：概率 P(·) 是满足以下条件的函数：
    • 非负性：P(A) ≥ 0
    • 规范性：P(Ω) = 1
    • 可列可加性：对互斥事件 A1, A2, ...，有 P(∪Ai) = ΣP(Ai)
    • 核心性质（由公理推导，必须熟记）：
    • P(Ø) = 0
    • 有限可加性：A， B互斥，则 P(A∪B) = P(A) + P(B)
    • P(Ā) = 1 - P(A)
    • 减法公式：P(A-B) = P(A) - P(AB) （特别地，若 B⊂A，则 P(A-B)=P(A)-P(B)）
    • 核心！加法公式：P(A∪B) = P(A) + P(B) - P(AB)

    常考

    P(A∪B∪C) = P(A)+P(B)+P(C) - P(AB)-P(AC)-P(BC) + P(ABC)

    1. 单调性：若 A⊂B，则 P(A) ≤ P(B)

    3. 条件概率（Key Concept！）¶

    • 定义：在事件B已发生的条件下，事件A发生的概率。 P(A|B) = P(AB) / P(B)， 要求 P(B) > 0。
    • 本质：样本空间的缩小。新的样本空间就是B。

    易错点

    1. P(A|B) 与 P(B|A) 完全不同！贝叶斯公式就是连接它们的桥梁。
    2. 条件概率本身也是概率，满足概率的所有公理和性质（如 P(Ā|B)=1-P(A|B)）。

    4. 三大公式¶

    1. 乘法公式（由条件概率定义直接推出）： P(AB) = P(A)P(B|A) = P(B)P(A|B)
    2. 推广：P(A1A2...An) = P(A1)P(A2|A1)P(A3|A1A2)...P(An|A1A2...An-1)

    3. 应用场景：涉及“分步发生”、“同时发生”且非独立的事件。

    4. 全概率公式（由因推果）：

    5. 前提：事件组 B1, B2, ..., Bn 是样本空间Ω的一个划分（即互斥且并集为Ω）。

    6. 公式：P(A) = Σ_{i=1}^{n} P(Bi)P(A|Bi)

    7. 直观理解：事件A可以通过多种不同的“原因”Bi导致发生，总的概率就是各种原因导致A发生的概率之和。

    易错点:划分的完备性必须检查！

    B1∪...∪Bn = Ω 且 Bi∩Bj=Ø。

    1. 贝叶斯公式（执果索因）：

    2. 公式：P(Bj|A) = [P(Bj)P(A|Bj)] / [Σ_{i=1}^{n} P(Bi)P(A|Bi)]

    3. 直观理解：在知道结果A发生的情况下，推断是由哪个原因Bj引起的可能性有多大。P(Bj)称为先验概率，P(Bj|A)称为后验概率。
    4. 核心考点：与全概率公式结合考查。分母就是全概率公式的 P(A)。

    5. 独立性¶

    • 定义：事件A与B独立 ⇔ P(AB) = P(A)P(B)。
    • 若 P(B)>0，也等价于 P(A|B) = P(A)。即B的发生不影响A的概率。
    • 性质：
    • 若A与B独立，则A与B̄、Ā与B、Ā与B̄都独立。
    • 互斥 vs 独立（极易混淆！）：
      • 若 P(A)>0, P(B)>0：
      • 若A, B互斥，则 P(AB)=0 ≠ P(A)P(B)>0，所以它们不独立。
      • 若A, B独立，则 P(AB)>0，所以它们不互斥。
      • 结论：对于概率为正的事件，互斥与独立不可能同时成立（除非概率为0）。互斥是集合关系，独立是概率关系。
    • 多个事件的独立性：
    • 三个事件A, B, C相互独立，必须同时满足：
      1. P(AB) = P(A)P(B)
      2. P(AC) = P(A)P(C)
      3. P(BC) = P(B)P(C)
      4. P(ABC) = P(A)P(B)P(C)

    常考反例

    前三条叫“两两独立”。两两独立 ≠ 相互独立。

    三、重点题型与解题套路¶

    1. 复杂事件的概率计算：

    步骤：① 用字母表示事件；② 用集合运算描述目标事件；③ 利用概率公式计算。

    技巧：善用对立事件 P(A)=1-P(Ā)， 尤其是当“至少一个”出现时。

    1. “分步”或“顺序”问题（使用乘法公式）：

    例如：不放回抽签、零件依次装配等。

    1. “由多个原因导致一个结果”问题（使用全概率公式）：

    题干常出现：“……来自三个不同的工厂/渠道”，“……由三种原因导致”。

    1. “已知结果，反推原因”问题（使用贝叶斯公式）： 题干常出现：“已知检测呈阳性，问真正患病的概率是多少？”，“产品不合格，问它是甲车间生产的概率是多少？”。

    2. 独立性判断与应用： 判断：严格用定义 P(AB)=P(A)P(B) 检查。 应用：若事件独立，则求交事件的概率可简化为概率的乘积。

    四、 查漏清单¶

    回答以下问题，检验你是否真的掌握了：

    1. 基础：P(A)=0.5, P(B)=0.6, P(A∪B)=0.8， 你能立刻求出 P(AB) 和 P(A-B) 吗？

    2. 理解：P(A|B) 和 P(B|A) 的区别是什么？能举个例子吗？

    3. 辨析：A, B互斥，且 P(A)>0, P(B)>0，它们独立吗？为什么？

    4. 公式：写出A, B, C三个事件全都不发生的概率表达式。

    5. 流程：面对一个贝叶斯问题（如诊断问题），你的解题步骤是怎样的？

    点击查看答案
    1. P(AB)=0.3, P(A-B)=0.2
    2. P(ĀB̄C̄) = 1 -P(A∪B∪C)， 再用三事件加法公式展开） 5.步骤：① 定义事件，如H=患病，T=阳性；② 写出已知条件 P(H), P(T|H), P(T|H̄)；③ 用全概求 P(T)；④ 用贝叶斯求P(H|T)）

    花20分钟把这些基础概念、关系和公式在白纸上整理一遍，形成你自己的逻辑图。这是你后续学习随机变量、统计推断的坚实地面。
