    随机变量的数字特征¶

    好的，现在我们进入随机变量的数字特征。这部分是概率论的“统计摘要”，用几个关键的数值来刻画随机变量的核心性质。它是连接概率论和数理统计的桥梁。

    一、 核心逻辑与知识地图¶

    核心思想： 用确定的数字来概括随机变量分布的主要特征。

    逻辑主线： 位置特征（数学期望）→ 离散程度（方差）→ 相互关系（协方差与相关系数）

    这些特征分为两类：

    1. 单个随机变量的特征：期望、方差。
    2. 两个随机变量间关系的特征：协方差、相关系数。

    二、 核心知识点精讲与易错点¶

    1. 数学期望（Expectation，均值）¶

    • 定义：随机变量取值的加权平均，权重为概率。
      • 离散型：E(X) = Σ [x_k * p_k]（要求级数绝对收敛）。
      • 连续型：E(X) = ∫_{-∞}^{+∞} x * f(x) dx（要求积分绝对收敛）。
    • 物理意义：分布的重心位置、长期平均。
    • 核心性质（必须熟记，用于简化计算）：
      1. 线性性（最重要）：E(aX + bY + c) = aE(X) + bE(Y) + c。
        无论X与Y是否独立都成立！
      2. 若 X ≥ 0，则 E(X) ≥ 0。
      3. 若 X 与 Y 独立，则 E(XY) = E(X)E(Y)。反之不成立。
    • 易错点：
      • 求 E(X) 时，离散型要覆盖所有 x_k，连续型积分要遍及整个定义域。
      • E(X^2) 是 X² 的期望，不是 [E(X)]²。

    2. 方差（Variance）¶

    • 定义：衡量随机变量取值相对于其均值的平均波动程度。 D(X) = Var(X) = E[ (X - E(X))² ]
    • 计算方差的核心公式（更常用）： D(X) = E(X²) - [E(X)]² 推导：D(X) = E(X² - 2X·E(X) + [E(X)]²) = E(X²) - 2E(X)E(X) + [E(X)]² = E(X²) - [E(X)]²
    • 性质：
      1. D(c) = 0，c是常数。
      2. D(aX + b) = a² D(X)。注意：加常数b不影响方差，乘系数a要平方。
      3. 若 X 与 Y 独立，则 D(X ± Y) = D(X) + D(Y)。
        • 更一般地：若 X1, ..., Xn 两两独立，则 D(Σa_iX_i) = Σa_i² D(X_i)。
    • 标准差：σ(X) = √D(X)，与 X 量纲相同。
    • 易错点：
      • 方差公式 E(X²) - [E(X)]² 要求先分别计算 E(X²) 和 E(X)，再相减。
      • 独立性是方差可加的必要条件。如果不独立，D(X+Y) = D(X)+D(Y)+2Cov(X,Y)。

    3. 协方差（Covariance）¶

    • 定义：衡量两个随机变量协同变化的程度。 Cov(X, Y) = E[ (X - E(X)) (Y - E(Y)) ]
    • 计算协方差的核心公式： Cov(X, Y) = E(XY) - E(X)E(Y) （这是最常用的计算公式，与方差公式 D(X)=E(X²)-[E(X)]² 类似）
    • 性质：
      1. Cov(X, Y) = Cov(Y, X)。
      2. Cov(aX, bY) = ab Cov(X, Y)。
      3. Cov(X+Y, Z) = Cov(X, Z) + Cov(Y, Z)。
      4. Cov(X, X) = D(X)。
      5. 若 X 与 Y 独立，则 Cov(X, Y) = 0。反之不成立。
    • 与方差的关系： D(X ± Y) = D(X) + D(Y) ± 2Cov(X, Y)

    4. 相关系数（Correlation Coefficient）¶

    • 定义：将协方差标准化，消除量纲影响，纯粹反映线性相关程度。 [ \rho_{XY} = \frac{Cov(X, Y)}{\sqrt{D(X)} \sqrt{D(Y)}} ]
    • 性质（核心）：
      1. |ρ_{XY}| ≤ 1。
      2. |ρ_{XY}| = 1 的充要条件是：存在常数 a, b（a≠0），使得 P{Y = aX + b} = 1。即 X 与 Y 以概率1存在严格的线性关系。
        • ρ = 1：完全正线性相关。
        • ρ = -1：完全负线性相关。
      3. ρ_{XY} = 0 时，称 X 与 Y 不相关。
    • 不相关 vs 独立（重中之重！）：
      • 独立 ⇒ 不相关（因为独立则 Cov=0，从而 ρ=0）。
      • 不相关 ⇏ 独立。不相关（ρ=0）仅表示没有线性关系，但可能有其他非线性关系。
      • 特例：对于二维正态分布，不相关 ⇔ 独立。
    • 易错点：
      • 相关系数衡量的是线性相关程度，而非任何相关。即使 ρ=0，X 和 Y 也可能有很强的曲线关系（如 Y=X²，当 X 对称分布时，ρ=0）。

    三、 重要公式与定理¶

    1. 方差计算公式：D(X) = E(X²) - [E(X)]²
    2. 协方差计算公式：Cov(X, Y) = E(XY) - E(X)E(Y)
    3. 随机变量和的方差： [ D(X+Y) = D(X) + D(Y) + 2Cov(X, Y) ] [ D(X-Y) = D(X) + D(Y) - 2Cov(X, Y) ]
    4. 切比雪夫不等式（了解）： 对任意 ε > 0，有 [ P{ |X - E(X)| ≥ ε } ≤ \frac{D(X)}{ε²} ] 它给出了随机变量偏离均值程度的概率上界。

    四、 常见分布的期望与方差（必须像公式表一样背熟）¶

    分布 记号 期望 E(X) 方差 D(X)
    0-1分布 B(1, p) p p(1-p)
    二项分布 B(n, p) np np(1-p)
    泊松分布 P(λ) λ λ
    均匀分布 U(a, b) (a+b)/2 (b-a)²/12
    指数分布 Exp(λ) 1/λ 1/λ²
    正态分布 N(μ, σ²) μ σ²

    推导技巧：很多分布（如二项、泊松）的期望方差可以通过定义或性质（如二项分布是n个独立的0-1分布之和）推导。

    五、 重点题型与解题套路¶

    1. 直接计算期望、方差：

      • 利用定义或公式。
      • 对于复杂函数 Y=g(X)，可用公式 E(Y)=∫ g(x)f(x)dx（连续型）或 Σ g(x_k)p_k（离散型）。不必先求Y的分布。

    2. 利用性质简化计算：

      • 见到 E(aX+bY+c)，直接用线性性拆开。
      • 见到 D(aX+b)，直接写 a²D(X)。
      • 求 Cov(X,Y)，优先用公式 E(XY)-E(X)E(Y)。

    3. 判断不相关与独立：

      • 判断独立：验证 f(x,y)=f_X(x)f_Y(y) 或 P(X=x,Y=y)=P(X=x)P(Y=y)。
      • 判断不相关：计算 Cov(X,Y) 或 ρ_{XY} 是否为0。
      • 特别注意：除非是二维正态，否则“不相关”不能推出“独立”。

    4. 综合题：结合具体分布，求期望、方差、协方差、相关系数，并讨论独立性。 经典套路：

      1. 先求边缘密度 f_X(x)， f_Y(y)。
      2. 求 E(X)， E(Y)， E(X²)， E(Y²)， E(XY)。
      3. 利用公式求 D(X)， D(Y)， Cov(X,Y)， ρ_{XY}。
      4. 判断独立性（验证 f(x,y)=f_X(x)f_Y(y)）。

    六、 终极查漏清单¶

    1. 公式：D(X) = E(X²) - [E(X)]² 和 Cov(X,Y) = E(XY) - E(X)E(Y) 的相似性你注意到了吗？
    2. 性质：E(XY) = E(X)E(Y) 成立，能推出 X 与 Y 独立吗？（不能！只能推出不相关）
    3. 计算：已知 X~U(0,1)， 求 E(X²) 和 D(X)。（答案：E(X²)=1/3， D(X)=1/12）
    4. 辨析：“独立”和“不相关”的区别与联系是什么？在什么条件下它们等价？
    5. 应用：对于二维均匀分布在一个圆形区域上，X 与 Y 是否独立？是否不相关？（通常不独立，但由于对称性，常常不相关 ρ=0）

    一个重要练习： 设 (X, Y) 的联合密度为 f(x, y) = x + y， 0 < x < 1, 0 < y < 1。

    1. 求 E(X)， E(Y)， Cov(X, Y)。
    2. 求 ρ_{XY}。
    3. X 与 Y 是否独立？是否不相关？
