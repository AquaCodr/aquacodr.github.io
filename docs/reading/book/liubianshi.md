# 月亮与六便士 （9.8-9.13）

**用自己喜欢的方式度过短暂的一生……**

> 或许他们深知，常人大凡都不能做自己想要做的那种人，而只能做他们不得不做的那种人。
>
> 作家更关心的是了解，而不是评判。
>
> 但最残忍的还是，这件事对别人并没有什么影响，世界照样运转，没有人会因为这个悲剧而活得更糟。
>
> 我们每个人在这个世界上都是独立的个体，每个人都被囚禁在一座铁塔里，只能靠一些符号交流，而这些符号并没有共同的价值，因此他们的意义是模糊而不确定的。
>
> 做自己想做的事，生活在自己喜欢的环境里，享受内心的宁静，这就是糟蹋生活吗？当一个有名的外科医生，每年赚得上万英镑，娶个漂亮妻子，这就是成功吗？
>
> 他就生活在那里，忘记了世界，世界也已把他遗忘。
>
> 世上只有很少的人能够努力追求并实现自己的理想。
>
> 他从来没有抱怨过自己的命运，也从未失去过勇气。
>
> 魔鬼总想引用《圣经》来达到自己的目的。

我没想到读这本书会如此快，6天，说实话，如果不是想要按照推荐书单来读的话，我自己是不会读这本书的。为什么？我知道这本书很有名，但从小到大的教育让我感到名著大多是很无趣的。关于一个男人抛妻弃子去画画的故事，歌颂理想嘛，呵呵，是啊，我都已经知道书里要讲什么了，剧透了，还有甚好读的呢。

但我错了，我低估了一本书能成为名著的门槛，这不只是一个故事那么简单，em...**这是一位作家深沉思考的结晶，我看见了矛盾，斗争，关于意义，还有人性，这是一本人生之书。**

刚看到史特利克兰抛妻弃子的时候，说实话，我想要写一篇长篇大论来骂他，骂他不负责任，根本就不配步入婚姻的殿堂，他是在骗婚，他毁掉了一个那么完美的家庭，就为了一个连自己都还不清楚是什么的理想么，死后天才的名头真的能抵得过生前美满幸福的生活么？

可我想做的事情故事里的“我”已经替我做了，“我”跑去骂过他，后来又与他直接或间接地产生了丝丝缕缕的联系，“我”得以断断续续地拼凑出他的一生。“我”厌恶过他，质疑过他，“我”反驳他亦反思自己，“我”最终理解了他：“连我自己都觉得奇怪，我描写的是一个冷酷，自私，粗暴，很不理性的人，可我却说他是个伟大的理想主义者。”
