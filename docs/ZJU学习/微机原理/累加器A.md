
> [!abstract] 概括
accumulator，是最核心、使用频率最高的寄存器，单片机大脑里的“头号工作台”。


它是一个8位寄存器，位于单片机中SFR区，地址E0H，是CPU进行算数运算和逻辑运算的主要场所
### 什么时候写A，什么时候写ACC？
**使用A**: 代表寄存器寻址，绝大多数是用A
**使用ACC**：典型场景
- PUSH ACC (压栈，不能写为 PUSH A )
- POP ACC (弹栈)
### 什么场景必须使用A，而不是其他寄存器？
1. MOVC 查表指令 (对存放于程序储存器中的数据表格进行查找传送)
2. MOVX 传送类指令 (与片外RAM传送数据)
3. 交换指令
