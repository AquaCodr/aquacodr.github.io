## 1.数据传送类指令 (29 条)
> [!abstract] 说明
> 传送类指令是最基本,使用最多的一类指令,主要用于数据的传送、保存以  及交换等场合。
1. MOV 类传送指令 (16条)    `格式:MOV <目的地址>, <源地址>`
	- 以累加器A为目的操作数的指令 (4条)
	- 以寄存器Rn为目的操作数的指令 (3条)
	- 以直接地址为目的操作数的指令 (5条)
	- 以间接地址为目的操作数的指令 (3条)
	- 16位立即数传送指令	  `MOV DPTR, #data16`
2. MOVC查表指令 (2条)
	`MOVC A, @A+DPTR `
	`MOVC A, @A+PC`
3. MOVX传送类指令 (4条)  
4. 堆栈操作类指令 (2条) 
	`PUSH direct`
	`POP direct`
> [!info]  功能
把**直接寻址**单元的内容传送到堆栈指针SP所指单元中
5. 交换指令 (5条)
## 2.算数运算类指令 (24 条)
> [!abstract]  说明
>主要完成加、减、乘、除四则运算,以及加1、减1、BCD码的运算和调整等
>多是基于寄存器A的
1. 加法指令ADD (4条)
2. 带进位加法指令ADDC (4条)
3. 带借位减法指令SUBB (4条)
4. 乘法指令MUL (1条)
5. 除法指令DIV (1条)
6. 自增指令INC (5条)
	分别是寄存器A，直接地址，寄存器，寄存器间接，DPTR
7. 自减指令DEC (4条)
	DPTR无减一指令，需要程序实现
8. 十进制调整指令 (1条)
	`DA A`  作用：将十六进制强行修正回十进制形式
## 3.逻辑操作类指令 (24 条)
> [!abstract]  说明
> 逻辑操作是按位进行的
1. 循环移位指令 (4条)
	分为带 C 位和不带 C 位两种，其中带 C 位左右移相当于乘或除以2
2. 求反指令 CPL (1条)
3. 清零指令 CLR (1条)
4. 逻辑与指令 (6条)
5. 逻辑或指令 (6条)
6. 逻辑异或指令 (6条)
## 4.控制转移类指令 (17 条)
> [!abstract]  说明
> 控制转移类指令用于控制程序的走向,故其作用区间是程序存储器
1. 无条件指令 (4条)
	分别是 LJMP、AJMP、SJMP、JMP
2. 条件转移指令 (8条)
	分别是JZ、JNZ、CJNE、DJNZ
3. 子程序调用及返回指令 (4条)
	分别是LCALL、ACALL、RET(普通返回)、RETI(中断返回)  注意压栈保护
> [!tip]  RET及RETI用法
如果在考试题目中看到一段程序以 `ORG 0003H`（外部中断入口）开始，那么结尾必须是 `RETI`。如果是你自定义的一个 `DELAY:` 子程序，结尾必须是 `RET`。
4. 空操作指令NOP (1条)
	摸鱼指令，用于延时1us
