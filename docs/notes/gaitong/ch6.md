# 统计量与抽样分布

好的，现在我们进入**样本及抽样分布**。这部分是数理统计的基石，回答了“我们如何用样本构造工具去推断总体”的问题。

---

## **一、 核心逻辑与知识地图**

**核心思想：** 从总体中抽取样本，构造不含未知参数的样本函数（统计量），研究这些统计量的分布（抽样分布），为统计推断（估计、检验）提供理论依据。

**逻辑主线：**
**总体** → **样本（随机、独立同分布）** → **统计量** → **抽样分布（特别是来自正态总体的三大分布）**

**核心难点：** 理解“统计量的分布”这一概念，并熟记正态总体下的三大抽样分布及其性质。

---

## **二、 核心知识点精讲与易错点**

### **1. 基本概念**

- **总体**：研究对象的全体。通常用一个随机变量 `X` 及其分布 `F(x; θ)` 表示。
- **样本**：从总体中随机抽取的 `n` 个个体 `(X₁, X₂, ..., Xₙ)`。要求：
    1.  **代表性**：每个 `X_i` 与总体 `X` 同分布。
    2.  **独立性**：`X₁, ..., Xₙ` 相互独立。
        **简称为：简单随机样本，独立同分布（i.i.d）**。
- **样本值/观测值**：样本的一次具体实现 `(x₁, x₂, ..., xₙ)`。注意区分**随机样本**（大写X，是随机变量）和**样本值**（小写x，是具体数）。

### **2. 统计量（核心）**

- **定义**：不依赖于任何未知参数的、样本的函数 `T = T(X₁, ..., Xₙ)`。
- **要点**：
    1.  统计量本身是**随机变量**（因为样本是随机的）。
    2.  不能包含未知参数（例如，如果 `μ` 未知，`(X̄ - μ)` 就不是统计量）。
- **常用统计量（必须熟记定义和性质）**：
    1.  **样本均值**：
        \[ \overline{X} = \frac{1}{n} \sum_{i=1}^{n} X_i \]
    性质：`E(X̄) = μ`， `D(X̄) = σ²/n`。
    2.  **样本方差**（**注意分母是 n-1**）：
        \[
        S^2 = \frac{1}{n-1} \sum_{i=1}^{n} (X_i - \overline{X})^2
        \]
    **重要性质**：`E(S²) = σ²`。即 `S²` 是 `σ²` 的**无偏估计**。这就是为什么分母是 `n-1` 而不是 `n`。
    （如果分母用 `n`，记为 `S_n^2`，则 `E(S_n²) = (n-1)σ²/n`，是有偏的。）
    3.  **样本标准差**：`S = √S²`。
    4.  **样本 k 阶原点矩**：`A_k = (1/n) Σ X_i^k`。
    5.  **样本 k 阶中心矩**：`B_k = (1/n) Σ (X_i - X̄)^k`。
        - 注意：`B₂ = (n-1)S²/n`，不是 `S²`。
- **顺序统计量**：将样本按从小到大排序 `X_(1) ≤ X_(2) ≤ ... ≤ X_(n)`。例如，`X_(1)` 是最小值，`X_(n)` 是最大值，中位数等。

### **3. 抽样分布**

- **定义**：统计量的概率分布。
- **研究意义**：因为统计量是随机变量，要进行统计推断（如构造置信区间、做假设检验），必须知道它的分布。
- **难点**：直接求一个统计量的精确分布通常很困难。但在**正态总体**的假设下，我们可以得到几个非常漂亮且重要的精确分布。

### **4. 单个正态总体的抽样分布（三大分布，重中之重！）**

设总体 `X ~ N(μ, σ²)`， `(X₁, ..., Xₙ)` 是来自该总体的样本，`X̄` 为样本均值，`S²` 为样本方差。

**必须熟记的结论：**

1.  **样本均值分布**：
    \[
    \overline{X} \sim N\left( \mu, \frac{\sigma^2}{n} \right)
    \]
    标准化得：
    \[
    \frac{\overline{X} - \mu}{\sigma / \sqrt{n}} \sim N(0, 1)
    \]
    **这是后续一切的基础。**

2.  **卡方分布 (χ² distribution)**：
    - 定义：若 `Z₁, ..., Z_k` 独立且均服从标准正态分布 `N(0,1)`，则 `χ² = Σ Z_i²` 服从自由度为 `k` 的卡方分布，记作 `χ² ~ χ²(k)`。
    - **性质**：`E(χ²(k)) = k`， `D(χ²(k)) = 2k`。
    - **与样本方差的关系（核心定理）**：
        \[
        \frac{(n-1)S^2}{\sigma^2} = \frac{\sum_{i=1}^{n} (X_i - \overline{X})^2}{\sigma^2} \sim \chi^2(n-1)
        \]
        **自由度是 n-1，不是 n**。这是因为 `Σ(X_i - X̄) = 0`，这 `n` 个偏差项有一个线性约束，导致独立变量只有 `n-1` 个。
    - 另外：`Σ[(X_i - μ)²/σ²] ~ χ²(n)`（因为 `(X_i - μ)/σ ~ N(0,1)`，且独立）。

3.  **t 分布 (Student’s t distribution)**：
    - 定义：若 `Z ~ N(0,1)`， `Y ~ χ²(k)`，且 `Z` 与 `Y` 独立，则
        \[
        T = \frac{Z}{\sqrt{Y/k}} \sim t(k)
        \]
    - **性质**：`t` 分布密度关于 `0` 对称，形状类似标准正态但尾部更厚。当 `k → ∞` 时，`t(k) → N(0,1)`。
    - **与样本均值的关系（核心定理，当 σ² 未知时用）**：
        由 `(X̄ - μ)/(σ/√n) ~ N(0,1)` 与 `(n-1)S²/σ² ~ χ²(n-1)`，且可以证明 **X̄ 与 S² 相互独立**（正态总体下的重要结论），可得：
        \[
        T = \frac{\overline{X} - \mu}{S / \sqrt{n}} \sim t(n-1)
        \]
        **这是 t 检验的理论基础。**

4.  **重要独立性定理（正态总体下）**：
    - `X̄` 与 `S²` 相互独立。
    - `X̄` 与每个偏差 `(X_i - X̄)` 都独立（从而与 `S²` 独立）。

### **5. 两个正态总体的抽样分布**

设 `X ~ N(μ₁, σ₁²)`， `Y ~ N(μ₂, σ₂²)`，样本分别为 `X₁, ..., X_{n₁}` 和 `Y₁, ..., Y_{n₂}`，且两个样本相互独立。记 `X̄`， `Ȳ`， `S₁²`， `S₂²` 分别为样本均值和样本方差。

1.  **均值差 `X̄ - Ȳ` 的分布**：
    \[
    \overline{X} - \overline{Y} \sim N\left( \mu_1 - \mu_2, \frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2} \right)
    \]
    标准化得：
    \[
    \frac{(\overline{X} - \overline{Y}) - (\mu_1 - \mu_2)}{\sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}}} \sim N(0,1)
    \]

2.  **F 分布 (Fisher–Snedecor distribution)**：
    - 定义：若 `U ~ χ²(m)`， `V ~ χ²(n)`，且 `U` 与 `V` 独立，则
        \[
        F = \frac{U/m}{V/n} \sim F(m, n)
        \]
    - **性质**：`F(m, n)` 与 `F(n, m)` 有关系：`F_{1-α}(m, n) = 1 / F_{α}(n, m)`。
    - **与样本方差比的关系（核心定理）**：
        由 `(n₁-1)S₁²/σ₁² ~ χ²(n₁-1)`， `(n₂-1)S₂²/σ₂² ~ χ²(n₂-1)`，且独立，可得：
        \[
        F = \frac{S_1^2 / \sigma_1^2}{S_2^2 / \sigma_2^2} = \frac{\sigma_2^2}{\sigma_1^2} \cdot \frac{S_1^2}{S_2^2} \sim F(n_1-1, n_2-1)
        \]
        特别地，若 `σ₁² = σ₂²`，则 `(S₁²/S₂²) ~ F(n₁-1, n₂-1)`。
        **这是方差比检验（F检验）的理论基础。**

3.  **两总体均值差的 t 分布（方差未知但相等）**：
    当 `σ₁² = σ₂² = σ²` 但未知时，定义**合并样本方差**：
    \[
    S_p^2 = \frac{(n_1 - 1)S_1^2 + (n_2 - 1)S_2^2}{n_1 + n_2 - 2}
    \]
    则：
    \[
    T = \frac{(\overline{X} - \overline{Y}) - (\mu_1 - \mu_2)}{S_p \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}} \sim t(n_1 + n_2 - 2)
    \]
    **这是两独立样本 t 检验（等方差）的理论基础。**

## **三、 重点题型与解题套路**

1.  **判断是否为统计量**：检查表达式中是否含有未知参数。
2.  **求统计量的数字特征**：利用期望、方差的性质，以及 `X̄` 和 `S²` 的已知性质。
    - 例如：求 `E(Σ(X_i - a)²)`，可拆成 `Σ E[(X_i - μ + μ - a)²]`。
3.  **确定抽样分布**：给定正态总体和统计量形式，判断它服从什么分布。
    - **关键步骤**：
        1. 看是否涉及均值 → 可能用到 `N(0,1)` 或 `t` 分布。
        2. 看是否涉及方差 → 可能用到 `χ²` 或 `F` 分布。
        3. 看参数是否已知：
            - `σ²` 已知，用 `Z = (X̄ - μ)/(σ/√n) ~ N(0,1)`。
            - `σ²` 未知，用 `T = (X̄ - μ)/(S/√n) ~ t(n-1)`。
        4. 涉及两个样本方差比，用 `F` 分布。
4.  **查分位数**：给定上侧分位数 `χ²_{α}(n)`、`t_{α}(n)`、`F_{α}(m, n)`，要会查表并利用对称性、倒数关系。
    - `t_{1-α}(n) = -t_{α}(n)`
    - `F_{1-α}(m, n) = 1 / F_{α}(n, m)`

## **四、 终极查漏清单**

1.  **概念**：为什么 `S²` 的分母是 `n-1` 而不是 `n`？（为了无偏性） `S²` 与总体方差 `σ²` 的关系是什么？
2.  **推导**：在正态总体下，为什么 `(n-1)S²/σ² ~ χ²(n-1)`？自由度为什么是 `n-1`？
3.  **辨析**：`X̄ ~ N(μ, σ²/n)` 成立的条件是什么？（总体正态，或 `n` 大时用 CLT 近似）如果总体不是正态分布，`(X̄ - μ)/(S/√n)` 还服从 `t(n-1)` 分布吗？（当 `n` 大时近似成立，精确分布不成立）
4.  **关系**：`t` 分布与 `F` 分布有什么关系？（`t²(k) ~ F(1, k)`）
5.  **应用**：从 `N(μ, σ²)` 中抽取样本 `n=16`，已知 `σ²=4`，求 `P(|X̄ - μ| < 1)`。（用 `Z = (X̄-μ)/(2/4) = 2(X̄-μ) ~ N(0,1)`，答案是 `2Φ(2)-1`）

**重要练习**：
设 `X₁, ..., Xₙ` 来自 `N(μ, σ²)`， `X̄` 和 `S²` 是样本均值和方差。判断下列统计量的分布：

1.  `(X̄ - μ) / (σ/√n)`
2.  `(X̄ - μ) / (S/√n)`
3.  `Σ(X_i - μ)² / σ²`
4.  `(n-1)S² / σ²`
5.  `[√n (X̄ - μ)]² / S²`
