# 一维随机变量及其概率分布

!!! note "笔记"
    从“事件思维”到“函数思维”的关键飞跃

## 一、逻辑主线：

**随机变量X** → **分布函数F(x)** （通用描述）→ **具体类型**

## **二、 核心知识点精讲与易错点**

### **1. 随机变量**

- **定义**：定义在样本空间Ω上的实值函数 `X = X(ω)`。
- **作用**：将随机事件数量化。例如，`{X ≤ a}` 表示一个事件。
- **分类**：根据取值方式分为**离散型**和**连续型**。

### **2. 分布函数（核心工具，统一描述）**

- **定义**：`F(x) = P{X ≤ x}`, `x ∈ (-∞, +∞)`。
- **四大性质（判定一个函数是否为分布函数的充要条件）**：
    1. **单调不减**：若 `x1 < x2`，则 `F(x1) ≤ F(x2)`。
    2. **右连续**：`F(x+) = F(x)`。**这是易错点！** 左连续不一定成立。
    3. **取值范围**：`0 ≤ F(x) ≤ 1`。
    4. **极限行为**：`lim_{x→-∞} F(x) = 0`； `lim_{x→+∞} F(x) = 1`。
- **重要公式**：
    - `P{a < X ≤ b} = F(b) - F(a)`
    - `P{X > a} = 1 - F(a)`
    - `P{X = a} = F(a) - F(a-)` （**对于连续型，此项恒为0**）
- **易错点**：`P{a ≤ X ≤ b} = F(b) - F(a-)`， 在连续型下等于 `F(b)-F(a)`，但在离散型下要小心左端点。

### **3. 离散型随机变量**

- **描述工具：分布律（列）**
    `P{X = x_k} = p_k, k=1,2,...`
    **性质**：`p_k ≥ 0`，且 `∑ p_k = 1`。
- **分布函数**：`F(x) = P{X ≤ x} = ∑_{x_k ≤ x} p_k`。是一个**右连续的阶梯函数**。

#### **4. 连续型随机变量**

- **描述工具：概率密度函数**
    - **定义**：存在非负可积函数 `f(x)`，使得 `F(x) = ∫_{-∞}^{x} f(t) dt`。
    - **性质**：
    1. `f(x) ≥ 0`
    2. `∫_{-∞}^{+∞} f(x) dx = 1` （归一性）
    3. **最重要的几何意义**：`P{a < X ≤ b} = ∫_{a}^{b} f(x) dx`， 即**概率是密度函数曲线下的面积**。
    - **与分布函数的关系**：在 `f(x)` 的连续点处，`F'(x) = f(x)`。
- **易错点（致命！）**：
    1. `f(x)` 的值**不是概率**！它可以大于1。只有对 `f(x)` 积分才是概率。
    2. 连续型随机变量取**任何单点值**的概率为0，即 `P{X = a} = 0`。因此，开区间和闭区间的概率相等：`P{a ≤ X ≤ b} = P{a < X ≤ b} = P{a < X < b} = P{a ≤ X < b}`。

#### **5. 常见分布（必须像九九乘法表一样熟记）**

| 类型     | 分布名称     | 记号       | 分布律/概率密度                             | 期望E(X)  | 方差D(X)    | 应用场景（关键记忆点）                                       |
| :------- | :----------- | :--------- | :------------------------------------------ | :-------- | :---------- | :----------------------------------------------------------- |
| **离散** | **0-1分布**  | `B(1, p)`  | `P(X=k)=p^k(1-p)^{1-k}`, k=0,1              | `p`       | `p(1-p)`    | 一次伯努利试验（成功/失败）                                  |
| **离散** | **二项分布** | `B(n, p)`  | `P(X=k)=C_n^k p^k (1-p)^{n-k}`, k=0,1,...,n | `np`      | `np(1-p)`   | **n重独立**伯努利试验的成功次数                              |
| **离散** | **泊松分布** | `P(λ)`     | `P(X=k)=(λ^k e^{-λ})/k!`, k=0,1,...         | `λ`       | `λ`         | 稀有事件在一定时间/空间内的发生次数。**二项分布当n很大p很小时可用λ=np的泊松分布近似** |
| **连续** | **均匀分布** | `U(a, b)`  | `f(x)=1/(b-a)`, a≤x≤b                       | `(a+b)/2` | `(b-a)²/12` | “等可能”落在区间内                                           |
| **连续** | **指数分布** | `Exp(λ)`   | `f(x)=λe^{-λx}`, x≥0                        | `1/λ`     | `1/λ²`      | 寿命、等待时间。**无记忆性**：`P(X>s+t | X>s)=P(X>t)`        |
| **连续** | **正态分布** | `N(μ, σ²)` | `f(x)=1/(√(2π)σ) e^{-(x-μ)²/(2σ²)}`         | `μ`       | `σ²`        | 大量独立同分布随机变量和的分布。**标准化**：若`X~N(μ,σ²)`，则`Z=(X-μ)/σ ~ N(0,1)` |

**正态分布特辑**：

- **标准正态分布 `N(0,1)`**：密度函数 `φ(x)`，分布函数 `Φ(x)`。
- **性质**：`Φ(-x) = 1 - Φ(x)`； `P{|X|<a} = 2Φ(a) - 1`。
- **3σ准则**：`P{|X-μ|<σ}≈0.6827`, `<2σ>≈0.9545`, `<3σ>≈0.9973`。**常考！**

#### **6. 随机变量函数的分布（Y = g(X)）**

这是本章的计算难点，核心是**利用等价的随机事件进行转换**。

**解题总思路：** 先求 `Y` 的分布函数 `F_Y(y) = P{Y ≤ y} = P{g(X) ≤ y}`，然后通过 `X` 的分布求出这个概率，最后通过求导（连续型）或差分（离散型）得到 `Y` 的密度或分布律。

**离散型（简单）**：

- 列出所有 `X` 的取值 `x_k`， 计算对应的 `y_k = g(x_k)`。
- 如果 `g` 是**一一对应**，则 `P{Y=y_k} = P{X=x_k}`。
- 如果 `g` 不是一一对应（多对一），则需合并相同 `y_k` 值的概率：`P{Y=y} = ∑_{x: g(x)=y} P{X=x}`。

**连续型（重点和难点）**：

1. **公式法（仅当 `g` 严格单调）**：
    - 若 `y=g(x)` 单调可导，其反函数为 `x=h(y)`， 则 `Y` 的概率密度为：
     `f_Y(y) = f_X(h(y)) * |h'(y)|`， 其中 `y` 在 `g` 的值域内。
    - **关键**：一定要记得乘上**导数绝对值**，并确定 `y` 的新定义域。
2. **万能通法：分布函数法**（强烈建议掌握）：
    - **步骤**：
        ① 写出 `F_Y(y) = P{Y ≤ y} = P{g(X) ≤ y}`。
        ② 将 `{g(X) ≤ y}` 这个事件转化为 `X` 在某个区间 `I` 内的事件（可能需要讨论 `y` 的范围）。
        ③ 利用 `X` 的分布（积分 `f_X(x)` 或计算 `F_X(x)`）求出这个概率，得到 `F_Y(y)`。
        ④ 对 `F_Y(y)` 求导，得到概率密度 `f_Y(y)`。
    - **优点**：普适，不易出错，尤其适合 `g` 非单调（如 `Y=X²`）、分段等情况。

## **三、 重点题型与解题套路**

1. **已知分布律/密度，求分布函数**：
    - 离散型：求和。
    - 连续型：积分。注意分段讨论。
2. **已知分布函数，求密度函数或概率**：
    - 密度：在可导点求导。
    - 概率：用 `F(b)-F(a)` 等公式。
3. **确定待定常数**：
    - 利用**归一性**：`∑ p_k = 1` 或 `∫ f(x) dx = 1`。
4. **判断函数性质**：
    - 给你一个函数，判断它能否作为分布函数/密度函数？用四大性质或两条性质验证。
5. **求随机变量函数的分布（综合大题）**：
    - **黄金法则**：优先使用**分布函数法**。清晰地写清步骤：`F_Y(y) = ... = P{...} = ...`。
    - 对于单调的线性函数 `Y=aX+b`，可以直接用公式法快速得出：若 `X~N(μ,σ²)`， 则 `Y~N(aμ+b, a²σ²)`；若 `X~U(a,b)`， 则线性变换后仍是均匀分布。

## **四、 终极查漏清单**

1. **概念**：分布函数 `F(x)=P{X≤x}` 为什么是“右连续”而不是“左连续”或“连续”？
2. **计算**：已知 `X~N(1,4)`， 求 `P{0<X<2}`。 （思路：标准化 `Z=(X-1)/2`）
3. **辨析**：连续型随机变量在一点的概率为0，这个事件是不可能事件吗？（不是！这是理解连续型的关键）
4. **对比**：写出二项分布 `B(n,p)` 和泊松分布 `P(λ)` 的分布律。它们之间有什么关系？
5. **应用**：求 `Y=e^X` 的密度函数，已知 `X~N(μ,σ²)`。 （这就是“对数正态分布”，请用分布函数法完整推导一次）
