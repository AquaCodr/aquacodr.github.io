# 点估计

好的，现在我们进入**点估计**。这是数理统计中“用样本推断总体”的第一类核心问题。其核心思想是：用一个具体的数值（估计量）去估计未知参数。

---

## **一、 核心逻辑与知识地图**

**核心思想**：构造样本的函数（统计量）作为未知参数的估计，并评价其好坏。

**逻辑主线**：
**待估参数 θ** → **构造估计量的两种主要方法**（矩估计、最大似然估计）→ **评价估计量的四个标准**（无偏性、有效性、一致性、均方误差）

**关键**：区分**估计量**（随机变量，`θ̂ = θ̂(X₁,…,Xₙ)`）和**估计值**（具体数值，`θ̂(x₁,…,xₙ)`）。

---

## **二、 核心知识点精讲与易错点**

### **1. 矩估计法（Method of Moments, MOM）**

- **基本思想**：用**样本矩**替换**总体矩**（矩匹配）。
- **具体步骤**：
    1.  写出总体矩的表达式（通常包含待估参数）。
        - `k` 阶总体原点矩：`μ_k = E(X^k)`
        - `k` 阶总体中心矩：`ν_k = E[(X - μ₁)^k]`
    2.  写出对应的样本矩。
        - `k` 阶样本原点矩：`A_k = (1/n) Σ X_i^k`
        - `k` 阶样本中心矩：`B_k = (1/n) Σ (X_i - X̄)^k`
    3.  令总体矩 = 样本矩，建立方程（组）。
    4.  解方程（组），得到参数的矩估计量 `θ̂_MOM`。
- **优点**：简单、直观，计算方便。
- **缺点**：不唯一（可以用不同阶的矩），有时不够优（如未充分利用分布信息）。
- **易错点**：
    - 对于多个参数，需要建立与参数个数相同的方程。
    - 常用的是低阶矩（一阶、二阶）。对于常见分布，通常：
        - 一个参数：用 `E(X) = X̄`。
        - 两个参数：用 `E(X) = X̄` 和 `E(X²) = A₂` 或 `D(X) = B₂`。

### **2. 最大似然估计法（Maximum Likelihood Estimation, MLE）**

- **基本思想**：“已发生的事件概率应最大”，即选择使当前样本观测值出现概率最大的参数值。
- **核心概念**：
    - **似然函数**：
        - 离散型：`L(θ) = L(θ; x₁,…,xₙ) = ∏ P(X_i = x_i; θ)`
        - 连续型：`L(θ) = ∏ f(x_i; θ)`
            （`L(θ)` 是参数 `θ` 的函数，样本观测值固定）
    - **对数似然函数**：`ℓ(θ) = ln L(θ) = Σ ln f(x_i; θ)` （求和比连乘更容易处理）。
- **求解步骤**：
    1.  写出似然函数 `L(θ)`。
    2.  取对数得 `ℓ(θ)`（或直接处理 `L(θ)`）。
    3.  对 `θ` 求导（或求偏导），令导数为0：`∂ℓ/∂θ = 0`。
    4.  解方程得到**驻点** `θ̂_MLE`。
    5.  验证该驻点是最大值点（通常检查二阶导或边界）。
- **MLE的性质（大样本下优良）**：
    - 相合性（一致性）
    - 渐近正态性
    - 不变性：若 `η = g(θ)`，则 `η̂_MLE = g(θ̂_MLE)`。（**矩估计无此性质**）
- **易错点**：
    - 当导数为0无解时，需考虑边界或单调性。
    - 对于**均匀分布 `U(0, θ)`** 这类分布，似然函数不连续/不可导，需用定义直接求最大值（`θ̂_MLE = max{X_i}`）。
    - 注意参数空间（定义域）。

### **3. 估计量的评选标准**

评价一个估计量 `θ̂` 的好坏，主要看它在大量重复抽样下的表现。

1.  **无偏性（Unbiasedness）**
    - 定义：如果 `E(θ̂) = θ` 对一切可能的 `θ` 成立，则称 `θ̂` 是 `θ` 的无偏估计。
    - **意义**：估计量没有系统偏差。
    - **重要例子**：
        - `X̄` 是 `μ` 的无偏估计。
        - `S²` 是 `σ²` 的无偏估计（但 `S` 不是 `σ` 的无偏估计）。
        - 矩估计通常是无偏的（但不总是）。
        - 最大似然估计可能是有偏的（如正态分布方差的 MLE 是 `S_n²`，有偏）。
    - **有偏估计**：偏差 `Bias(θ̂) = E(θ̂) - θ`。

2.  **有效性（Efficiency）**
    - 定义：在**无偏估计类**中，方差越小越有效。
    - 比较：设 `θ̂₁` 和 `θ̂₂` 都是 `θ` 的无偏估计，如果 `D(θ̂₁) ≤ D(θ̂₂)` 对所有 `θ` 成立，且至少在某 `θ` 处严格不等，则称 `θ̂₁` 比 `θ̂₂` 有效。
    - **克拉默-拉奥下界（C-R下界）**：给出了无偏估计方差的理论下界。达到该下界的估计称为**有效估计**（最小方差无偏估计，MVUE）。

3.  **一致性（相合性，Consistency）**
    - 定义：随着样本量 `n → ∞`，估计量依概率收敛于真值。即对任意 `ε > 0`，`lim P(|θ̂ - θ| ≥ ε) = 0`。
    - **意义**：大样本下估计是准确的。
    - **判别法**：通常，如果 `θ̂` 是无偏的，且 `lim D(θ̂) = 0`，则它是一致估计。大多数合理的估计量（如矩估计、MLE）都具有相合性。

4.  **均方误差准则（MSE，综合权衡）**
    - 定义：`MSE(θ̂) = E[(θ̂ - θ)²]`。
    - **分解公式（非常重要）**：
        \[
        MSE(\hat{\theta}) = D(\hat{\theta}) + [Bias(\hat{\theta})]^2
        \]
        （方差 + 偏差的平方）
    - **意义**：综合衡量估计的精度。在比较**有偏估计和无偏估计**时，常用 MSE 标准（因为可能用较小的偏差换取方差的大幅降低，从而 MSE 更小）。

## **三、 重点题型与解题套路**

### **题型1：求矩估计**

- **套路**：
    1. 计算总体矩 `E(X)`， `E(X²)` 等（含参数）。
    2. 计算样本矩 `X̄`， `A₂` 等。
    3. 令相等，解方程。

**例**：总体 `X ~ Exp(λ)`， 求 `λ` 的矩估计。

- `E(X) = 1/λ`。
- 令 `1/λ = X̄` ⇒ `λ̂ = 1/X̄`。

### **题型2：求最大似然估计**

- **套路**：
    1. 写出似然函数 `L(θ)`（连乘）。
    2. 取对数 `ℓ(θ)`。
    3. 求导，令为0。
    4. 解出 `θ̂`，并讨论。

**例**：总体 `X ~ N(μ, σ²)`， 求 `μ`， `σ²` 的 MLE。

- `ℓ(μ,σ²) = -n/2·ln(2π) - n/2·ln(σ²) - Σ(x_i-μ)²/(2σ²)`
- 对 `μ` 求导 ⇒ `μ̂ = X̄`
- 对 `σ²` 求导 ⇒ `σ̂² = (1/n) Σ (X_i - X̄)² = S_n²` （有偏）

### **题型3：估计量的评价**

- **判断无偏性**：计算 `E(θ̂)`，看是否等于 `θ`。
- **比较有效性**：在无偏估计类中，比较方差。
- **讨论一致性**：看是否 `n→∞` 时，`θ̂` 依概率收敛（常利用大数定律或切比雪夫不等式）。
- **计算 MSE**：用公式 `MSE = 方差 + 偏差²`。

## **四、 终极查漏清单**

1.  **概念**：估计量和估计值的区别是什么？
2.  **计算**：对于均匀分布 `U(a, b)`，如何求 `a`， `b` 的矩估计和最大似然估计？（矩估计：`â = X̄ - √3·S`， `b̂ = X̄ + √3·S`；MLE：`â = min{X_i}`， `b̂ = max{X_i}`）
3.  **性质**：最大似然估计的“不变性”是什么意思？矩估计有这个性质吗？
4.  **公式**：MSE 的分解公式是什么？如果估计量是无偏的，MSE 等于什么？
5.  **辨析**：“一个好的估计量应该方差尽可能小”和“一个好的估计量应该无偏”，这两个要求哪个更重要？（取决于问题，通常希望 MSE 小，可以牺牲无偏性换取更小的方差）

**重要练习**：
设总体 `X` 服从参数为 `λ` 的泊松分布，`X₁,…,Xₙ` 为样本。

1.  用矩估计法和最大似然估计法分别求 `λ` 的估计量。
2.  判断这两个估计量是否无偏、是否一致。
3.  比较它们的有效性（方差）。
